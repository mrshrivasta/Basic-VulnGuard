## Educational Vulnerability Scanner

Basic VulnGuard is an educational vulnerability scanner developed by Karanam Shrivasta.

The project helps identify publicly disclosed vulnerabilities in Python, npm, and Ruby dependencies using public vulnerability databases such as OSV.dev and CVE sources.

The scanner provides dependency auditing, package lookups, local port scanning, security analysis, vulnerability reporting, CSV export, JSON export, and a modern Flask-based dashboard.

# 📦 Installation

```bash
git clone https://github.com/mrshrivasta/Basic-VulnGuard.git

cd Basic-VulnGuard

pip install flask packaging

python vuln_scanner.py
```

Open:

```text
http://localhost:5000
```

---

# 🖥️ CLI Usage

```bash
python vuln_scanner.py

python vuln_scanner.py --installed

python vuln_scanner.py --scan requirements.txt

python vuln_scanner.py --scan package.json

python vuln_scanner.py --scan Gemfile.lock

python vuln_scanner.py --pkg requests 2.18.0
```
